Investigacion de etiquetas

  1. Etiqueta B

    2. La B nos sirve para agregar un titulo a la paguina Web que estemo creando

  2. Etiqueta p

    3. Nos sieve para crear un texto a la paguina que estemos creando

  3. Etiqueta img

    4. La etiqueta img nos sirve para colocar imagenes adentro de la pagina que estemos creando

  4. Etiqueta center

    5. La etiqueta center nos ayuda a centrar el texto o todo los elementos que agregemos a la paguina

  5. Etiqueta table

    6. La etiqueta

    en HTML se usa para crear una tabla en una página web. Esta etiqueta es el contenedor principal para todo el contenido de la tabla

  6. Etiqueta tr

    7. La etiqueta tr en HTML define una fila dentro de una tabla. Se utiliza para estructurar las filas de una tabla,

  7. Etiqueta td

    8. define una celda de datos en una tabla

  8. Etiqueta H1

    9. se utiliza para definir el encabezado principal o título principal de una página web

  9. Etiqueta H2

    10. encargada de definir subtítulos o encabezados secundarios dentro de una página web

  10. Etiqueta H3

    11. cse utiliza para definir encabezados de tercer nivel en una página web

  11. Etiqueta H4

    12. define un encabezado de nivel 4 en una página web

  12. Etiqueta H5

    13. define un encabezado de quinto nivel en un documento HTML

  13. Etiqueta H6

    14. define el encabezado de nivel más bajo de un documento

  14. Etiqueta A

    15. se utiliza para crear hipervínculos o enlaces a otras páginas web, archivos, o ubicaciones dentro de la misma página

  15. Etiqueta HR

    16. se usa para insertar una línea horizontal en una página web, que sirve para separar contenido o crear divisores visuales
